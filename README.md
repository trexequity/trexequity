## What is Trex Equity?

AI agents now generate real revenue. API calls, subscription tiers, enterprise contracts. Some agents clear $50k–$500k per month at break-even. But there is no legal path to share that revenue with retail supporters.

Agent tokens today are unregistered securities with no dividend mechanism, no shareholder rights, and no secondary-market compliance. The SEC will come for them. It is only a matter of time.

**Trex Equity** solves this by providing a full-stack platform that:

- **Incorporates** the agent as an LLC or C-Corp (Delaware)
- **Files** Reg CF ($5M cap) or Reg D (accredited, unlimited) with the SEC
- **Mints** Token-2022 compliance-wrapped equity shares on Solana
- **Streams** agent revenue into auto-dividends paid quarterly in USDC
- **Powers** compliant secondary markets via transfer hooks

The result: the first **regulated AI-agent equity market** on Solana, and **$TREX** captures value from every deal.
